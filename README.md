ATM System Project
Description
The ATM System Project is a simple software application that simulates basic ATM functions. 
Users can log in using a PIN, check their account balance, withdraw and deposit money, and view transactions. 
The project uses a frontend and backend to demonstrate full-stack development.

Project Structure
ATM_SYSTEM/
├── ATM_Frontend/
├── ATM_System_backend/
└── README.md

Technologies Used
Frontend: CSS, TypeScript, React
Backend: Java
Database: MySQL
Tools: Git, GitHub

Features

PIN-based user login
Balance checking
Cash withdrawal and deposit
Transaction history
Secure backend processing

How to Run Frontend
cd ATM_Frontend
npm install
npm start

Backend
Open backend project in IntelliJ / Eclipse
Configure MySQL database
Run using Apache Tomcat
Purpose

This project is developed for learning full-stack development and understanding basic banking system operations.
